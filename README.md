# E-Mapp
E-Mapp is a message box application and has been designed with a simple interface. Notifies the sender of invalid e-mail addresses in mass mails.

## Usage

### 1-) Sign in with your Gmail account
<img width="303" alt="image" src="https://user-images.githubusercontent.com/67822910/96339451-6e37e980-109d-11eb-989d-6434115bc723.png">
<img width="303" alt="image" src="https://user-images.githubusercontent.com/67822910/96339491-bc4ced00-109d-11eb-813a-9c7e248224e4.png">

### 2-) Fill in the boxes and hit the send button

NOTE: If the Subject box is left blank, it will default to "No subject".
